# vk_messenger

**Это учебный проект. Задумка: написать мессенджер для ВК на Flutter.**

## Что использовалось:
**scoped_model** для управления состоянием

**GetIt** : ServiceLocator

**flutter_secure_storage** для хранения персональных данных


**VK Api** для серверного сообщения с ВК
